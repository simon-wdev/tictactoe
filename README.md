### Tic-Tac-Toe The Odin Project

[Play the game here!](https://simon-wdev.github.io/tictactoe/)

**Features:**
* play against a friend
* enter names for both players
* auto-checks for tie or win-conditions
* reset without reloading the whole page

**Tech:** Javascript (Vanilla), HTML/CSS

The goal was to write code without having global variables, so it is written in a module pattern. This was achieved by using `IIFE's (Immediately Invoked Function Expression)` as well as a `factory function` for creating the players.
